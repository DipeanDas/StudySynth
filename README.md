## 🎯 What is StudySynth?

StudySynth is an AI-powered web app that transforms your handwritten or digital notes into:

- 📄 **Concise summaries**
- 🧠 **Interactive quizzes**
- 🔊 **Audio explanations**

All from just a few uploaded images.


## ⚡ Features

- 🖼️ Upload up to 3 images of notes  
- ✍️ AI-generated structured summaries  
- 🧠 Difficulty-based quiz generation (Easy / Medium / Hard)  
- ✅ Instant quiz evaluation with scoring  
- 🔊 Clean audio narration 
- 🎨 Clean, modern UI


## 🧠 How it works

1. Upload your notes (images)  
2. Select quiz difficulty  
3. Click **Generate**  
4. Get:
   - Summary  
   - Audio explanation  
   - Quiz + score
